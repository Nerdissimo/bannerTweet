BannerTweet - Bannerizza il tweet!

Perché a volte un RT non basta. Trasforma un tweet in un bruttissimo banner!

HowTo

Aggiungi un hashtag in fondo all'URL con l'id del tweet da bannerizzare!
Puoi trovare l'ID nei dettagli del tweet (Espandi -> Dettagli), in fondo all'URL.

Hint: Puoi generare più banner nella stessa pagina accodando più ID separati da una virgola.

Questo tweet -> https://twitter.com/nerdissimo/status/274536056400015360
Può diventare -> http://bannertweet.nerdissimo.it/#274536056400015360
O accodando 2 ID -> http://bannertweet.nerdissimo.it/#274536056400015360,274599324384636928

Tools

Aggiungi questo link a preferiti -> BannerTweet
Vai nella pagina di dettaglio di un tweet (Espandi -> Dettagli) e clicca sul preferito per visualizzarne la versione banner!
